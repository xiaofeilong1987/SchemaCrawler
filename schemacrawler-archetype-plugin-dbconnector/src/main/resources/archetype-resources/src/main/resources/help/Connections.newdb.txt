--- Connection Options ---

-host=<host>
	Host name. Optional, defaults to localhost.
	
-port=<port>
	Port number. Optional, defaults to 9898.	

-database=<database>
	Database name.
	
-user=<user>
	Database user name.

-password=<password>
	Database password.
